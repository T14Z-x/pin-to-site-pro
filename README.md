# Pin-to-Site Demo (Next.js)

Drop a Google Maps link/pin or place text, then generate a quick one-page business website preview you can send to local businesses as a sales demo.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## What this demo includes

- Place parsing from Google Maps URL or plain text
- Category + tone presets (restaurant, dentist, hospital, salon, repair)
- Live iframe preview of generated site
- Copy generated HTML
- Download generated HTML file

## Verify

```bash
npm run lint
npm run build
```
